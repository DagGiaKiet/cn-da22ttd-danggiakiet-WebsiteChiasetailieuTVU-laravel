# Phần Mềm Liên Quan

Thư mục này chứa các phần mềm, công cụ liên quan trong quá trình thực hiện đồ án.

## Nội dung

- Công cụ hỗ trợ phát triển
- Plugin, extension cần thiết
- Library đặc biệt (không có trên package manager)
- Tools tự phát triển

## Lưu ý

- Chỉ chứa các phần mềm cần thiết, không phổ biến
- Có hướng dẫn cài đặt, sử dụng
- Ghi rõ version, nguồn gốc
- Tuân thủ bản quyền
