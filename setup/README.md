# Thư mục Setup

Thư mục này chứa các tập tin cần thiết để cài đặt và chạy ứng dụng.

## Nội dung

### 1. Database
- File SQL: `database_setup.sql` - Script tạo cơ sở dữ liệu và dữ liệu mẫu
- Hướng dẫn import database trong file `SETUP_DATABASE.md`

### 2. Môi trường
- File `.env.example` - Template cấu hình môi trường
- Hướng dẫn cấu hình trong file `SETUP_ENVIRONMENT.md`

### 3. Hướng dẫn cài đặt
Xem file `INSTALLATION_GUIDE.md` để biết chi tiết các bước cài đặt đầy đủ.

## Yêu cầu hệ thống
- PHP >= 8.1
- Composer
- MySQL >= 5.7
- Node.js & NPM
- Web Server (Apache/Nginx)
