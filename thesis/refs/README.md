# Tài Liệu Tham Khảo

Thư mục này chứa các tài liệu, văn bản tham khảo trong quá trình thực hiện đồ án.

## Nội dung

- Sách, giáo trình liên quan
- Bài báo khoa học
- Tài liệu kỹ thuật
- Luận văn, đồ án tham khảo

## Quy ước đặt tên

Tên file theo định dạng:
- `[Tác giả]_[Năm]_[Tên tài liệu rút gọn].pdf`
- Ví dụ: `NguyenVanA_2023_Laravel_Best_Practices.pdf`

Hoặc tên gợi nhớ dễ hiểu:
- `Laravel_Documentation.pdf`
- `MySQL_Performance_Guide.pdf`
