# Tài Liệu Đồ Án (HTML)

Thư mục này chứa tài liệu đồ án dạng web (HTML).

## Nội dung

- Phiên bản web của báo cáo đồ án
- Tài liệu hướng dẫn sử dụng online
- Demo tài liệu tương tác (nếu có)

## Lưu ý

- File HTML nên có cấu trúc rõ ràng
- Bao gồm CSS, JavaScript cần thiết
- Đảm bảo hiển thị tốt trên các trình duyệt
