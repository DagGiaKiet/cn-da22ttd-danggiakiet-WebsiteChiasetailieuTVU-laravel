# Tài Liệu Báo Cáo Đồ Án

Thư mục này chứa các tài liệu báo cáo, trình bày đồ án.

## Nội dung

- File PowerPoint (.ppt, .pptx) - Slide trình bày
- File Video (.avi, .mp4) - Video demo sản phẩm
- Poster đồ án (.pdf, .jpg)
- Tài liệu tóm tắt (abstract)

## Lưu ý

- Slide trình bày nên súc tích, rõ ràng
- Video demo không quá 10 phút
- Poster theo đúng template của trường
